# DB1100

Disse dokumentene kan bli brukt til å lære bedre hvordan en setter opp databaser og litt om spørringer. 
De kan også bli brukt til referanse til senere prosjekter, men det er ikke tillatt å direkte kopiere Database Script eller andre materialer som jeg har lagt ut.
